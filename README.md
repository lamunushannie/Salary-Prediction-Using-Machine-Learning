# Salary-Prediction-Using-Machine-Learning
Machine Learning project that builds and trains a Multiple Linear Regression model to help HR departments predict candidate salaries using experience, test performance, and interview evaluation data
Project Overview
This project builds a simple Machine Learning model to predict a candidate’s salary based on:
•	Years of Experience
•	Written Test Score
•	Interview Score
The goal is to help an HR department estimate salaries for new candidates using past hiring data.
What This Project Does
•	Loads hiring data
•	Cleans and prepares the data
•	Trains a Multiple Linear Regression model
•	Predicts salary for new candidates
Tools Used
•	Python
•	Pandas
•	Scikit-learn
